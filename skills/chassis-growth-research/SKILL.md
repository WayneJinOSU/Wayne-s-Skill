# 底盘型成长股研究

## Overview

用于分析“传统业务有现金流和利润底盘，同时因 AI、数据中心、新能源、机器人、先进制造等新周期出现第二成长曲线”的制造业公司。

核心问题：

```text
这家公司是否具备旧业务托底、新业务抬天花板、平台能力可复用、扩产融资能承接、未来利润能消化当前估值的特征？
```

## When to use

适合研究 A 股制造业二次成长公司，例如电力电子、PCB、高速连接、线缆、光通信、散热/液冷、新能源车零部件、工控自动化、机器人零部件、半导体设备/材料、先进封装材料、AI 服务器零部件。

不适合直接用于银行、地产、保险、纯消费品牌、纯资源周期、早期创新药、纯互联网软件、没有旧业务底盘的概念股。

## Workflow

1. 先判断是否适用框架。
   - 必须同时具备存量业务底盘、新成长曲线、平台能力复用或承接动作。
2. 按七层结构分析。
   - 底盘、天花板、平台复用、承接动作、成长阶段、利润桥、远期估值。
3. 把公司归类。
   - 预期型、半兑现型、兑现型，或不符合框架。
4. 建立利润桥。
   - 未来利润 = 存量业务利润 + 新业务新增利润 - 扩张期费用/折旧/财务成本/爬坡损耗。
5. 用当前市值反推远期 PE。
   - 重点看 2026E、2027E、2028E 利润能否消化当前估值。
6. 最后列证伪点。
   - 用 3 到 5 个最可能推翻逻辑的变量约束结论。

## Output format

输出时优先按下面结构：

1. 一句话结论：预期型、半兑现型、兑现型，或不符合框架。
2. 公司底盘：主营、利润、现金流、客户、行业地位、下限。
3. 新成长曲线：新业务、产业趋势、天花板、产业链位置。
4. 平台复用：技术、客户、工艺、产能、供应链、认证体系。
5. 承接动作：扩产、融资、海外基地、设备、客户验证、量产。
6. 兑现阶段：为什么属于当前阶段。
7. 利润桥：存量利润 + 新业务利润 - 扩张成本。
8. 估值表：当前市值对应 2025、2026E、2027E、2028E PE。
9. 证伪点：最关键的 3 到 5 个。
10. 最终判断：机会、风险、适合什么类型资金。

## Data discipline

分析具体公司时，优先使用最新公告、年报/季报、交易所互动、公司官网、投资者关系纪要和可靠市场数据。涉及当前市值、股价、PE、业绩预告、订单、扩产、融资等可变信息时，需要核对日期并说明口径。
