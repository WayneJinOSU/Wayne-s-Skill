# 承接动作研究

## Purpose

用于回答：公司是否用真金白银和经营动作承接新业务预期？

## Inputs

优先读取扩产公告、募投项目、定增/H股/可转债、资本开支、在建工程、设备采购、海外基地、客户验证、订单、量产进度、研发投入和人员变化。

## Workflow

1. 汇总已经发生的承接动作。
2. 区分资本开支、融资、客户验证、订单、量产和研发投入。
3. 判断动作是实质承接还是口径表述。
4. 检查动作和新业务收入/利润是否匹配。
5. 说明动作如何传导到收入和利润，避免把融资、扩产本身当作兑现。
6. 列出尚未验证的反证点。
7. 给出落地等级：强承接、初步承接、弱承接、无承接。

## Output

```text
承接结论：
已发生动作：
尚未验证事项：
收入利润匹配度：
动作到业绩的传导：
反证或替代解释：
落地风险：
对主线判断的影响：
```
