# 地方商业银行投研框架

## 1. 核心判断

地方商业银行研究的核心不是“净利润涨了多少”，而是判断：

- 利润是不是来自主业息收
- 有没有靠卖债、公允价值、少提拨备把利润做漂亮
- 资产质量和区域信用有没有被市场低估
- 当前股价是在交易基本面改善，还是交易悲观预期修复

## 2. 利润桥

优先把净利润拆成：

```text
净利润变化 =
净利息收入变化
+ 手续费及佣金净收入变化
+ 投资收益变化
+ 公允价值变动变化
+ 汇兑/其他非息变化
- 业务及管理费变化
- 信用减值损失变化
- 所得税变化
```

### 质量判断

- 净利息收入增长：高质量，优先看负债成本是否下降
- 贷款扩张带来息收增长：中高质量，前提是资产质量没恶化
- 投资收益增长：谨慎，可能不可持续
- 公允价值收益增长：更谨慎，受市场波动影响大
- 少提拨备增厚利润：低质量
- 压费用增厚利润：短期有效，长期有限

## 3. 净利息收入拆解

```text
净利息收入 =
生息资产规模 × 资产收益率
- 付息负债规模 × 负债成本率
```

### 资产端

- 贷款增速是否领先行业
- 对公、按揭、消费贷、小微贷结构
- 债券投资占比是否过高
- 高收益资产是否在减少

### 负债端

- 存款增长是否稳定
- 活期存款占比是否下滑
- 定期化压力是否缓解
- 存款成本是否下降
- 是否过度依赖同业负债

## 4. 资产质量

不要只看不良率和拨备覆盖率，还要一起看：

- 关注类贷款比例
- 逾期贷款比例
- 不良生成率
- 核销和转出规模
- 信用减值损失
- 拨备覆盖率
- 贷款拨备率
- 房地产、城投、制造业、小微等重点暴露

### 红旗

- 净利润增长，但信用减值损失明显下降
- 不良率稳定，但关注类和逾期持续上升
- 拨备覆盖率高，但贷款拨备率和不良生成率恶化

## 5. 债券投资

单独拆四个科目：

- 投资收益
- 公允价值变动损益
- 其他综合收益 OCI
- 其他债权投资

### 判断方法

- 投资收益大增：可能是卖债兑现收益
- 投资收益大降：利润可能更依赖主业
- OCI 大幅为负：债券估值拖累净资产
- OCI 转正：债券估值修复

## 6. 区域 beta

地方银行必须结合区域看：

- 当地 GDP
- 财政收入
- 土地出让
- 房地产压力
- 城投压力
- 产业结构
- 人口流入/流出
- 地方政府支持能力
- 大股东背景

## 7. 估值与分红

常用判断顺序：

1. PB 是否仍在低位
2. ROE 是否有韧性
3. 分红是否稳定
4. 股息率是否足够提供底部保护
5. 市场是否已经把坏消息提前定价

## 8. 建议输出模板

每次输出尽量包含：

- 一句话结论
- 利润桥
- NII 拆解
- 资产质量判断
- 债券投资影响
- 区域 beta
- 估值/分红结论
- 最关键的上行和下行风险
