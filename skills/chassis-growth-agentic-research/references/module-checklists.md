# 内置模块问题清单

本文件承接已删除的成长股子模块框架。主 skill 不再依赖独立子 skill，但正式研究必须按这些清单执行；如果数据不足，写“证据不足”和缺口，而不是省略模块。

## 旧业务底盘

目的：回答如果没有第二成长曲线，公司还能不能活得不错；旧业务能否提供收入、利润、现金流、客户基础和估值下限。

优先读取：最新年报、季报、业务分部、毛利率、经营现金流、应收账款、存货、客户结构和行业排名。

工作流：

1. 拆主营业务和收入结构。
2. 判断旧业务收入是否稳定，利润是否可持续。
3. 检查毛利率、费用率、净利率和经营现金流。
4. 检查应收账款、存货、合同负债和回款质量。
5. 判断客户基础、行业地位和竞争壁垒。
6. 说明证据如何推导到“下限”判断，并列出反证或替代解释。
7. 给出下限判断：硬底盘、一般底盘、弱底盘、无底盘。

输出字段：

```text
主营和收入结构：
盈利与现金流：
客户和行业地位：
推理链：
反证或替代解释：

```

## 第二成长曲线

目的：回答新业务是否真的把公司从小池塘带到大池塘，是否能抬高未来收入和利润天花板。

优先读取：公司新业务披露、行业数据、客户应用场景、产品代际变化、单机价值量、市场空间和竞争格局。

工作流：

1. 定义新业务和对应的新周期。
2. 判断新业务属于 AI、数据中心、新能源、机器人、先进制造、半导体或其他方向。
3. 估算行业空间、增速和行业阶段。
4. 判断公司处于核心环节还是边缘环节。
5. 判断单机价值量、单位价值量或份额是否提升。
6. 区分新业务是抬收入、抬利润率、抬估值身份，还是三者兼具。
7. 列出反证或替代解释，避免把行业空间直接等同于公司利润。
8. 给出天花板判断：显著抬高、有限抬高、只是概念、不成立。

输出字段：

```text

行业空间和阶段：
公司产业链位置：
价值量变化：
利润中枢推理：
反证或替代解释：
关键不确定性：
对主线判断的影响：
```

## 平台复用能力

目的：回答新业务是不是靠原有能力自然切入，而不是硬蹭概念。

优先读取：公司原有产品技术、客户结构、工艺路线、产线/产能、供应链、认证体系、研发团队和新产品说明。

工作流：

1. 列出旧业务的核心能力。
2. 列出新业务需要的关键能力。
3. 逐项匹配技术、客户、工艺、产能、供应链、认证体系。
4. 判断是否存在完全跨界、能力断层或概念包装。
5. 说明复用链条如何降低新业务兑现难度，并列出替代解释。

## 承接动作

目的：回答公司是否用真金白银和经营动作承接新业务预期。

优先读取：扩产公告、募投项目、定增/H 股/可转债、资本开支、在建工程、设备采购、海外基地、客户验证、订单、量产进度、研发投入和人员变化。

工作流：

1. 汇总已经发生的承接动作。
2. 区分资本开支、融资、客户验证、订单、量产和研发投入。
3. 判断动作是实质承接还是口径表述。
4. 检查动作和新业务收入/利润是否匹配。
5. 说明动作如何传导到收入和利润，避免把融资、扩产本身当作兑现。
6. 列出尚未验证的反证点。
7. 给出落地等级：强承接、初步承接、弱承接、无承接。

## 利润桥

目的：回答新故事能不能转化为未来利润中枢，未来利润能否消化当前估值。

优先读取：历史利润、业务分部毛利率、费用率、产能、订单、价格、出货量、资本开支、折旧、研发费用、财务费用和管理层指引。

固定公式：

```text
未来利润 =
存量业务利润
+ 新业务新增利润
- 扩张期费用/折旧/财务成本/爬坡损耗
```

工作流：

1. 建立存量业务利润底盘。
2. 估算新业务收入：TAM、份额、价值量、产能、订单或出货。
3. 估算新业务利润率：毛利率、净利率、规模效应、议价能力。
4. 估算扩张成本：研发、折旧、人员、销售、财务、汇兑和爬坡损耗。
5. 说明从当期利润到远期利润的跃迁路径和最敏感变量。
6. 列出会让利润桥失效的反证条件。
7. 做保守、中性、乐观三种情景。

## 远期估值与证伪

目的：回答当前市值到底是在低估未来、合理定价，还是已经透支 2 到 3 年利润。

优先读取：最新股价、总市值、股本、2025 利润、2026E-2028E 利润假设、同行估值、历史估值和关键证伪指标。

工作流：

1. 确认当前股价和总市值，并注明日期和来源。
2. 用当前市值反推 2025、2026E、2027E、2028E PE。
3. 判断利润假设属于保守、中性还是乐观。
4. 判断估值状态：低估、合理、偏贵、透支。
5. 说明当前市值隐含的是哪一年、哪种利润情景，避免只给 PE 表不解释含义。

## 成长阶段判定

总控在整合时必须判断公司处于哪一阶段：

| 类型 | 特征 | 股价弹性 | 主要风险 |
| --- | --- | ---: | --- |
| 预期型成长股 | 新业务刚被市场发现，利润尚未明显体现 | 最大 | 证伪风险高 |
| 半兑现型成长股 | 新业务已放量，市场开始验证持续性 | 中高 | 业绩波动杀估值 |
| 兑现型成长股 | 业绩已经大幅兑现，市场充分定价 | 较低 | 见顶或估值压缩 |
| 不符合框架 | 没有底盘或没有真实新曲线 | 不适用 | 题材退潮 |

判断时重点看预期差是否还在，而不是只看基本面好坏。
