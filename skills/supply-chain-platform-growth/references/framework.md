# 供应链平台型成长股研究框架

## 1. 定位

这个框架研究的不是所有成长股，而是大产业周期中的“关键供应链平台”：

- 下游处在 AI 算力、数据中心、新能源、机器人、先进封装、半导体设备/材料等强产业周期。
- 公司不是最高技术定义者，但承担关键制造、交付、材料、设备、连接、散热、电源或配套角色。
- 公司具备客户认证、复杂产品量产、规模交付、全球产能、供应链组织或良率成本优势。
- 订单、排产、出货和财报利润能验证利润中枢上移，而不只是概念或扩产。

核心判断句：

```text
产业周期够大 + 供应链位置关键 + 订单出货领先 + 利润平台换台阶 + 当前估值未过度透支
```

## 2. 分析链条

正式分析按这条链推进：

```text
产业大趋势
-> 客户资本开支
-> 供应链地位
-> 订单/排产/出货
-> 收入确认
-> 利润中枢上移
-> 市场身份重估
-> 股价重估或估值透支
```

供应链平台股不是单纯靠“技术垄断”涨，也不是简单靠“代工扩量”涨。它的核心是：在产业周期爆发时，市场重新定价公司的大规模交付能力和供应链组织能力。

## 3. 适用范围

适合：

- AI 服务器、GPU/ASIC 服务器、云厂商算力硬件交付平台。
- 高速 PCB、HDI、IC 载板、服务器主板、交换机相关供应链。
- 液冷散热、电源、连接器、高速线缆、光模块、光器件。
- 先进封装设备/材料、半导体设备/材料关键配套。
- 消费电子、通信设备、工业制造公司切入 AI 硬件、机器人、新能源等大周期。

不适合：

- 银行、地产、保险、纯资源周期、纯消费品牌。
- 纯软件互联网、早期创新药、生物科技。
- 无收入利润底盘、只有概念没有订单和产能承接的题材股。
- 主要靠商品价格周期，而不是交付放量和供应链平台能力的公司。

如果公司只是普通代工，且没有客户认证、交付规模、复杂产品量产、供应链组织或全球产能优势，必须降级为一般制造公司。

## 4. 六层结构

### 第一层：产业周期是否足够大

问：下游需求是否真实爆发，并且仍有未来 1 到 3 年可见度？

看：

- 客户资本开支是否扩张。
- 新产品平台是否提升价值量、复杂度或 ASP。
- 服务器、交换机、PCB、液冷、电源、连接器、光模块等环节是否同步放量。
- 需求是新增、替换、海外扩张，还是一次性拉货。
- 供给扩张是否已经带来价格和利润率压力。

这层决定大方向。没有产业周期，供应链平台重估很难持续。

### 第二层：市场身份是否切换

问：市场是否正在把公司从旧身份切到新身份？

常见路径：

```text
消费电子/通信设备/普通制造
-> AI 服务器/高速网络/算力硬件供应链平台
```

状态分类：

| 状态 | 特征 | 估值含义 |
| --- | --- | --- |
| 未切换 | 市场仍按旧业务和普通制造定价 | 估值弹性有限 |
| 正在切换 | 订单、利润或客户信号推动身份重估 | 弹性最大 |
| 已充分切换 | 市场已按新身份定价，需要利润继续验证 | 风险转向杀估值 |

身份切换必须由订单、出货、利润率、客户结构或市场定价行为共同验证。不要把“AI 概念”直接等同于“身份切换成功”。

### 第三层：供应链地位是否关键

问：公司是核心平台、关键供应商、重要配套，还是普通供应商？

看：

- 客户认证和合作深度。
- 复杂产品量产和良率。
- 全球产能、交付规模和供应链组织能力。
- 多产品、多客户、多区域扩散。
- 替代难度、替代周期和竞争对手扩产。
- 是否能分享价值量提升，而不是只增收被压价。

供应链平台公司的壁垒不一定是芯片 IP 或品牌心智，也可以来自认证、规模、交付、良率、成本和全球组织能力。

### 第四层：订单出货是否领先财报

问：增长是否已经在订单、排产、出货中出现，并且能传导到收入和利润？

看：

- 明确订单、量产、出货放量、客户项目。
- 海关/产业链数据、排产、产能利用率。
- 收入确认节奏和财报滞后程度。
- 毛利率、净利率是否改善。
- 存货、应收、合同负债、经营现金流是否支持增长质量。

订单信号分级：

| 信号 | 含义 |
| --- | --- |
| 强信号 | 明确订单金额/数量、量产、出货放量、财报利润验证 |
| 中信号 | 扩产、设备、客户认证、合作框架、产能爬坡 |
| 弱信号 | 概念、传闻、笼统进入供应链、未量化表述 |

### 第五层：利润中枢是否上移

问：公司是否从旧利润平台切到新利润平台？

季度利润年化只能作为压力测试：

```text
Q4 x 4 不是精确预测，而是检验公司是否进入新利润平台的情景假设。
```

必须回答：

- 高利润季度是偶发，还是新平台起点？
- 下一年订单能否支撑全年高利润？
- 后年是否还能继续增长，而不是下一年见顶？
- 收入增长是否带来毛利率、净利率或规模效应改善？
- 现金流、应收、存货是否支持利润质量？

利润桥：

```text
未来利润 =
旧平台利润
+ 新周期新增利润
- 扩张费用/折旧/财务成本/爬坡损耗
```

### 第六层：估值是否透支

问：当前股价反映的是哪一年、哪一种利润情景？

常见定价切换：

```text
当年利润定价
-> 下一年利润定价
-> 后年利润定价
-> 增速放缓后回到当年利润定价
-> 杀估值
```

必须用当前市值反推 2025、2026E、2027E、2028E PE，并说明利润假设属于保守、中性还是乐观。如果只有乐观情景才能消化估值，不能写成便宜。

## 5. 成长阶段

| 类型 | 特征 | 股价弹性 | 主要风险 |
| --- | --- | ---: | --- |
| 预期型供应链成长 | 身份刚切换，订单和利润尚未充分体现 | 最大 | 证伪风险高 |
| 半兑现型供应链成长 | 订单和利润部分体现，市场继续上修未来 | 中高 | 业绩波动杀估值 |
| 兑现型供应链成长 | 利润大幅兑现，市场已经充分乐观 | 下降 | 增速放缓和估值压缩 |
| 不符合框架 | 无真实订单、无平台地位或无利润底盘 | 不适用 | 题材退潮 |

判断时重点看预期差是否还在，而不是只看公司基本面好坏。

## 6. 常见误判

- 把行业景气误判成公司壁垒。
- 把扩产误判成订单。
- 把订单误判成利润。
- 把收入增长误判成利润中枢永久上移。
- 把季度高利润误判成全年常态。
- 把 AI 概念误判成市场身份切换成功。
- 把供应链关键地位拔高成顶级科技垄断。
- 忽略应收、存货和现金流恶化。
- 忽略客户压价、竞争扩产和估值锚回落。

## 7. 报告必须回答的问题

- 公司是否符合供应链平台型框架，还是只是一家普通制造/代工/配套公司？
- 市场正在交易什么：身份切换、订单放量、利润中枢上移、估值透支，还是错杀修复？
- 产业周期是否仍在上行，客户 capex 是否能支撑未来 1 到 3 年？
- 公司在供应链中处于什么位置，壁垒和可替代性如何？
- 订单出货是否真实放量，财报是否已确认，增长质量是否可靠？
- 利润平台是否换台阶，未来利润桥是否能走通？
- 当前市值反映到哪一年利润，是否已经透支？
- 哪些证据会推翻主线逻辑？
