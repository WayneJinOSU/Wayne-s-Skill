# 原成长股框架保留说明

本 skill 保留底盘型成长股判断框架。变化发生在执行机制：从“一次性按 checklist 汇总”，改成“多角色研究 + 中间文件 + 反方审查 + 投资人写作”。

## 核心判断句

```text
旧业务托底 + 新业务抬天花板 + 平台能力可复用 + 扩产融资能承接 + 未来利润能消化当前市值
```

这句话不能被改写成普通的“成长性好、估值合理”。它包含五个递进问题：

- 没有新业务，公司有没有收入、利润、现金流和客户基础。
- 新业务是否把公司带到更大的市场或更高利润率的环节。
- 新业务是否复用旧业务能力，而不是硬跨界。
- 公司是否已经用扩产、融资、客户验证、订单、量产或研发投入承接预期。
- 未来利润中枢是否足以消化当前市值。

## 七层结构

### 第一层：底盘是否够硬

问：如果没有新业务，公司还能不能活得不错？

看收入稳定性、利润、经营现金流、客户基础、行业地位、业务分散度。这层决定下限。没有底盘就是纯题材股。

### 第二层：新赛道是否抬高天花板

问：新业务是不是把公司从小池塘带到大池塘？

看 TAM、未来 3 到 5 年增速、行业阶段、技术路线、产业链位置、单机价值量或单位价值量。这层决定上限。

### 第三层：平台复用能力

问：它是不是靠原有能力自然切进去？

看技术、工艺、客户、供应链、产能、认证体系是否复用。这层决定可信度。完全跨界要显著降权。

### 第四层：承接动作

问：公司有没有真金白银押注新方向？

看扩产公告、资本开支、定增、H 股、可转债、项目贷款、海外基地、关键设备采购、客户验证、量产进度、新产品代际升级、研发投入和人员扩张。这层决定预期能不能落地。

### 第五层：成长阶段

| 类型 | 特征 | 股价弹性 | 主要风险 |
| --- | --- | ---: | --- |
| 预期型成长股 | 新业务刚被市场发现，利润尚未明显体现 | 最大 | 证伪风险高 |
| 半兑现型成长股 | 新业务已放量，市场开始验证持续性 | 中高 | 业绩波动杀估值 |
| 兑现型成长股 | 业绩已经大幅兑现，市场充分定价 | 较低 | 见顶或估值压缩 |

判断时重点看预期差是否还在，而不是只看基本面好坏。

### 第六层：利润桥

```text
未来利润 =
存量业务利润
+ 新业务新增利润
- 扩张期费用/折旧/财务成本/爬坡损耗
```

拆四块：

- 存量业务利润：传统业务能贡献多少稳定利润。
- 新业务收入：TAM、份额、价值量、产能、订单或出货。
- 新业务利润率：毛利率、净利率、规模效应、议价能力。
- 扩张期成本：研发、折旧、人员、销售、财务、汇兑、爬坡损耗。

这层决定未来利润中枢。

### 第七层：远期估值

不要只看当前 PE，要用当前市值反推未来 PE：

```text
远期 PE = 当前总市值 / 未来归母净利润
```

核心不是算出一个 PE，而是回答：当前市值到底在提前定价哪一年、哪一种利润情景；如果只有乐观情景才能消化估值，就不能写成便宜。

## 论证原则

正式报告不是填满七层结构，而是把结论论证出来。每个重要判断都要使用：

```text
证据：公告、财报、行情或产业事实是什么
推理：这些事实为什么支持底盘、天花板、复用、承接、利润或估值判断
反证：哪些事实可能削弱这个判断，或者是否存在更保守的解释
判断：在证据和反证之间，结论应该给到什么强度
```

不能因为出现扩产、订单、募投、概念或市值倒挂，就直接跳到“利润会兑现”或“估值低估”。必须解释中间机制：收入如何变利润、产能如何变出货、客户验证如何变复购、股权价值如何变安全垫、费用和现金流如何影响利润质量。

## 报告必须回答的问题

- 公司属于预期型、半兑现型、兑现型，还是不符合框架。
- 如果没有第二成长曲线，公司还能不能维持合理收入、利润、现金流和客户关系。
- 新业务到底是什么，处在什么产业阶段和产业链位置。
- 新业务是抬收入、抬利润率、抬估值身份，还是三者兼具。
- 新业务是否自然复用旧业务的技术、客户、工艺、产能、供应链或认证体系。
- 公司是否用融资、扩产、研发、设备、客户验证、订单或量产动作承接新预期。
- 从当期利润到远期利润的跃迁靠什么实现。
- 当前市值是在低估未来、合理定价，还是已经透支 2 到 3 年利润。
- 哪些变量最可能推翻主线逻辑。
- 公司适合什么类型资金，最大机会和最大风险分别是什么。
