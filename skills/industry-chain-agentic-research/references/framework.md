# 行业产业链研究框架

## 核心判断链

```text
行业定义 -> 产业链位置 -> 需求/供给/价格 -> 利润池 -> 竞争格局
-> 财务映射 -> 估值预期 -> 风险证伪
```

不要把“行业空间大”直接写成“上市公司有投资价值”。必须证明空间如何变成收入、利润、现金流和估值。

## 行业类型分诊

| 类型 | 重点问题 | 典型证据 |
| --- | --- | --- |
| 资源周期 | 价格、成本曲线、库存、供给约束 | 商品价格、库存、产能、开工率、成本曲线 |
| 制造加工 | 产能、良率、成本、客户认证、价格战 | 产能利用率、毛利率、客户、产品良率、订单 |
| 资本开支链 | 下游 capex、订单、交付、收入确认 | 招标、订单、排产、合同负债、设备交付周期 |
| 消费品牌 | 价格、渠道、库存、复购、品牌力 | 批价、渠道库存、动销、客单价、费用投放 |
| 平台渠道 | 用户、流量、抽佣、网络效应 | DAU/MAU、ARPU、GMV、take rate、留存 |
| 监管行业 | 政策、牌照、价格管制、资本约束 | 政策文件、牌照数量、定价规则、资本充足率 |
| 技术路线驱动 | 路线切换、替代风险、研发兑现 | 技术路线图、客户验证、专利、研发投入、量产节奏 |

## 七个模块

### 1. 行业定义与边界

必须回答：

- 研究的是大行业、子行业，还是具体产业链环节。
- 卖的是什么产品或服务。
- 谁买单，解决什么需求。
- 哪些看似相近但商业模式不同的行业应排除。

### 2. 产业链结构与公司位置

必须回答：

- 上游、中游、下游、终端需求分别是什么。
- 公司处在哪个环节。
- 上下游谁有议价权。
- 风险压在哪个环节，利润留在哪个环节。

### 3. 需求、供给与价格机制

必须回答：

- 需求来自新增、替换、渗透率、资本开支、政策、出口还是库存周期。
- 供给是否容易扩张，建设周期多长，退出难不难。
- 价格由供需、成本、招投标、品牌、渠道、政策或长期合同决定。
- 当前处在景气上行、半山腰、顶部、下行、底部还是出清阶段。

### 4. 利润池与竞争格局

必须回答：

- 哪个环节毛利率、净利率、ROIC、现金流最好。
- 利润池是否正在迁移。
- 集中度、进入门槛、替代风险和客户切换成本如何。
- 龙头是在扩份额、稳份额，还是被新进入者削弱。

建议输出表：

| 环节 | 收入规模 | 毛利率/净利率 | ROIC/现金流 | 议价权 | 周期性 | 主要风险 |
| --- | --- | --- | --- | --- | --- | --- |

### 5. 财务映射

强制写出传导链：

```text
行业变量 -> 产品价格/销量/成本/产能利用率
-> 毛利率/费用率 -> 利润 -> 现金流 -> 估值
```

必须说明：

- 哪些变量抬收入。
- 哪些变量抬毛利率。
- 哪些变量只抬收入但不抬利润。
- 哪些变量会增加应收、存货、折旧、财务费用或资本开支。

### 6. 估值与市场预期

必须回答：

- 当前市场在定价景气早期、半山腰、顶部，还是底部修复。
- 行业估值应该用 PE、PB、PS、EV/EBITDA、P/FCF，还是资产/订单/储量等口径。
- 当前估值是否只在乐观情景下成立。
- 行业好、公司好、股票好是否被混为一谈。

### 7. 风险与跟踪指标

必须输出：

- 3 到 7 个关键证伪点。
- 未来需要跟踪的高频和低频指标。
- 哪些指标先于财报变化。

## 强判断格式

所有核心结论必须使用：

```text
证据：
推理：
反证/替代解释：
判断强度：强 / 中强 / 中 / 弱
```

## 常见误区

- 把 TAM 大写成公司利润大。
- 把需求增长写成公司收入增长，忽略供给扩张。
- 把产品进入热门赛道写成利润率提升。
- 只看收入规模，不看利润池和现金流。
- 只看龙头收入最大，不看 ROIC 和自由现金流。
- 只用低 PE 判断便宜，不解释利润所处周期位置。
