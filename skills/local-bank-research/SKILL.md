# 地方商业银行投研

## Overview

用于研究地方商业银行股，不只看净利润增速，而是判断利润来自主业息收、投资兑现、拨备调节还是会计波动，并结合区域信用、分红和市场预期差判断股价。

## When to use

在分析城商行、农商行、区域性银行、银行年报/季报、股息率重估、区域信用风险、债券投资波动或大股东行为时使用。

## Workflow

1. 先看股价和预期。
   - 判断市场原本在交易什么：业绩超预期、资产质量改善、高股息重估、错杀修复，还是债券、拨备、非息扰动。
2. 再拆利润桥。
   - 必看：净利息收入、手续费及佣金、投资收益、公允价值变动、其他非息、业务及管理费、信用减值损失、所得税。
3. 再拆净利息收入。
   - 分成资产端规模与收益率、负债端规模与成本率两侧看。
   - 重点判断增长是来自规模扩张、负债成本下降，还是高收益资产占比变化。
4. 再验资产质量。
   - 同时看不良率、关注类、逾期、核销、不良生成率、拨备覆盖率、贷款拨备率和重点行业暴露。
5. 单独看债券投资。
   - 区分投资收益、公允价值变动、OCI、其他债权投资，判断利润是不是被卖债或估值波动放大或压低。
6. 评估区域 beta。
   - 结合当地 GDP、财政、地产、城投、人口、产业结构、政府支持能力和大股东背景。
7. 最后落到估值与分红。
   - 看 PB、ROE、分红稳定性、股息率和市场是否已提前定价悲观预期。

## Output format

输出时优先按下面结构组织：

1. 结论先行：一句话判断这只银行股在交易什么。
2. 证据分解：利润桥、NII、资产质量、债券投资、区域 beta。
3. 估值判断：当前定价是偏贵、合理还是仍有预期差。
4. 风险提示：最可能推翻判断的 1 到 3 个变量。
