# 底盘型成长股研究

## Overview

用于分析“传统业务有现金流和利润底盘，同时因 AI、数据中心、新能源、机器人、先进制造等新周期出现第二成长曲线”的制造业公司。

核心问题：

```text
这家公司是否具备旧业务托底、新业务抬天花板、平台能力可复用、扩产融资能承接、未来利润能消化当前估值的特征？
```

## When to use

适合研究 A 股制造业二次成长公司，例如电力电子、PCB、高速连接、线缆、光通信、散热/液冷、新能源车零部件、工控自动化、机器人零部件、半导体设备/材料、先进封装材料、AI 服务器零部件。

不适合直接用于银行、地产、保险、纯消费品牌、纯资源周期、早期创新药、纯互联网软件、没有旧业务底盘的概念股。

## Workflow

把这个 skill 当作“主工作流调度器 + 子工作流深挖器”使用。不要直接跳到最终结论；必须先完成每个子工作流的小结，再合成最终报告。

1. 主工作流：适用性判断。
   - 判断公司是否适合该框架；如果只是部分适用，要说明哪些部分适用、哪些部分不适用。
2. 子工作流 A：底盘业务。
   - 检验旧业务是否提供收入、利润、现金流、客户基础和下限支撑。
3. 子工作流 B：第二成长曲线。
   - 检验新周期是否显著抬高公司天花板。
4. 子工作流 C：平台复用。
   - 检验新业务是否自然延伸自原有技术、客户、工艺、供应链、产能或认证体系。
5. 子工作流 D：承接动作。
   - 检验公司是否已有资本开支、融资、扩产、设备采购、客户验证、订单或量产动作。
6. 子工作流 E：成长阶段。
   - 判断公司处于预期型、半兑现型、兑现型，还是不符合框架。
7. 子工作流 F：利润桥。
   - 估算未来利润：存量业务利润 + 新业务新增利润 - 扩张期成本。
8. 子工作流 G：远期估值。
   - 用当前市值反推 2025、2026E、2027E、2028E PE。
9. 子工作流 H：证伪点。
   - 找出最可能推翻投资逻辑的 3 到 5 个变量。
10. 最终合成。
   - 汇总所有子工作流结论，判断公司是在讲故事、正在兑现，还是已经充分定价。

## Output format

输出时优先按下面结构：

1. 一句话结论：预期型、半兑现型、兑现型，或不符合框架。
2. 子工作流结论表：逐项列出 A-H 的结论、证据强度、主要不确定性。
3. 公司底盘：主营、利润、现金流、客户、行业地位、下限。
4. 新成长曲线：新业务、产业趋势、天花板、产业链位置。
5. 平台复用：技术、客户、工艺、产能、供应链、认证体系。
6. 承接动作：扩产、融资、海外基地、设备、客户验证、量产。
7. 兑现阶段：为什么属于当前阶段。
8. 利润桥：存量利润 + 新业务利润 - 扩张成本。
9. 估值表：当前市值对应 2025、2026E、2027E、2028E PE。
10. 证伪点：最关键的 3 到 5 个。
11. 最终判断：机会、风险、适合什么类型资金。

## Data discipline

分析具体公司时，优先使用最新公告、年报/季报、交易所互动、公司官网、投资者关系纪要和可靠市场数据。涉及当前市值、股价、PE、业绩预告、订单、扩产、融资等可变信息时，需要核对日期并说明口径。
