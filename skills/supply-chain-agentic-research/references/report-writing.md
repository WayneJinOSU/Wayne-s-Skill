# 投资人写作规范

本文件只负责正式研报正文的写作风格：如何把中间研究写成普通投资人能读懂、但仍有信息密度和判断强度的文本。终稿结构、扩写蓝图、复杂度 profile 和反摘要闸门由 [final-report.md](final-report.md) 负责；QA 与出版清理由 [qa-gates.md](qa-gates.md) 负责；市场变量总姿态由 [research-posture.md](research-posture.md) 负责；估值和三表接力边界由 [handoffs.md](handoffs.md) 负责。

## 写作目标

- `final_report` 的写法是主笔重构，不是模块结论拼接。正文要把行业、技术、竞争、客户、产能、订单和利润桥重排成一条投资人能跟住的叙事链。
- 先讲公司在产业链里做什么、凭什么赚钱，再讲市场为什么重新关注它。
- 先讲市场正在交易的变量，再讲哪些已经验证、哪些只是市场口径或情景假设。
- 先解释技术路线、产品代际或材料等级如何改变价值量、ASP、用量、毛利率或认证门槛，再讲公司在哪个环节受益。
- 先讲订单如何变收入、收入如何变利润、利润如何变现金流，再给最终判断。
- 表格只能做索引、对比和数字承载；关键推理必须用段落讲清楚。

## 正文叙事顺序

写作时优先使用下面的叙事链，而不是机械复述模块顺序：

```text
公司做什么
-> 市场正在交易什么
-> 产业周期或技术路线为什么改变需求/价值量/价格
-> 公司为什么能分到增量
-> 订单、产能、原材料和客户认证如何影响收入、毛利、费用、折旧和现金流
-> 乐观变量成立时利润斜率如何变陡
-> 变量不成立时逻辑降级到哪一档
-> 后续用什么信号验证或证伪
```

每个核心章节都要服务于这条链。若一段内容不能解释市场变量、价值量、利润桥、证伪点或跟踪指标，就降为背景信息，不抢主线。

## 市场变量段落

高重要市场变量不能只写“仍需验证”或“后续关注”。成文时按下面的句式展开：

```text
变量：市场正在交易什么。
变化来源：新增需求、供需缺口、价格周期、产品代际、客户认证、产能窗口或原材料传导。
公司受益环节：材料、零部件、设备、制造、系统、渠道、控股子公司或参股公司。
财务传导：量、价、毛利率、份额、费用、折旧、良率、现金流、少数股东权益或投资收益。
证据属性：公告事实、一手资料、结构化数据、市场/券商口径、合理推断或情景假设。
变量增强时：收入、ASP、毛利率、订单稳定性、现金流或利润中枢如何变化。
变量减弱时：逻辑降级成周期修复、补库涨价、低端放量、扩产先行、成本挤压或主线证伪。
验证指标：下一季或下一年看什么。
```

证据等级写清楚后，仍要继续写判断边界。证据不足不是删除变量的理由；高重要变量应降级保留为市场口径、情景假设或待验证线索，并写清成立和不成立分别意味着什么。

## 机制写法

技术路线和产品代际要写成传导链：

```text
下游平台迭代
-> 性能瓶颈变化
-> 零部件/材料规格升级
-> 单机/单柜/单车/单套价值量、ASP、毛利率或用量变化
-> 供应商认证、份额、良率、产能约束
-> 订单、收入、利润和现金流兑现
```

原材料和价格链要写清：涨价来自供需、成本、库存还是稀缺；公司能否顺价；顺价滞后、库存重估和客户压价如何影响毛利率和现金流。

产能和订单不能直接写成利润。必须把 `订单 -> 排产 -> 出货 -> 收入确认 -> 利润释放 -> 现金回收` 讲完整；没有数据的环节写成验证缺口和情景边界。

利润桥要写成桥，不是结论表：

```text
未来利润 =
存量供应链利润
+ 新周期新增利润
- 扩张费用/折旧/财务成本/爬坡损耗
```

控股子公司要处理少数股东权益；参股公司要处理权益法收益；周期高点利润不能直接当长期中枢。

## 证据边界语言

- 公告、财报和交易所披露写成事实。
- 客户/行业一手资料写清来源和适用范围。
- 券商模型、产业链口径和市场传闻写成市场口径或情景假设。
- 合理推断要写条件，不要伪装成事实。
- 强判断要带条件：什么发生时升级，什么缺失时降级。

不要用保守措辞把正文压成摘要。正确写法是：先把变量成立时的利润弹性讲清，再说明证据等级、验证路径和降级条件。

## 标题和语言

- 标题要有判断密度，优先写成“变量 + 传导 + 结论方向”。少用“分析、梳理、空间较大、有望、仍需验证”这类缓冲标题。
- 普通投资人第一次看到术语时，要用一句话解释。
- 每个数据都要服务于主线、利润桥或证伪点；不要为了显得深而堆材料。
- 对外正式正文不要出现主控 skill、agent 阶段、中间产物、提示词、工具名或内部审稿语言。
- `final_report` 不写目标价、目标市值、PE/PEG/SOTP、买卖建议或“当前贵不贵”的估值结论；相关变量只进入研究阶段 readiness、终稿 gate 后 handoff 或后续独立估值产物。

## 必须避免

- 用概念词替代解释：AI 算力、平台化、国产替代、全球交付、第二曲线。
- 用“仍需验证、后续关注、证据不足”替代判断。
- 把行业景气直接写成公司壁垒。
- 把扩产写成订单。
- 把订单写成利润。
- 把收入增长写成利润平台永久上移。
- 把季度高利润年化写成全年确定预测。
- 把券商/产业链口径写成公告事实。
- 把并表利润、权益法收益、少数股东损益重复计算。
