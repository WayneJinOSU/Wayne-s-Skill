# Model Difference Matrix

聚合层只做 PEG 与 DCF 的差异提示，不重算、不调参、不裁判哪个模型正确。

## Four Quadrants

| 组合 | 含义 | 重点解释 | 跟踪指标 |
| --- | --- | --- | --- |
| PEG 高、DCF 高 | 成长和现金流同步验证 | 哪些收入/利润率/现金流假设共同成立 | 分业务收入、毛利率、OCF/UFCF、Capex/revenue |
| PEG 高、DCF 低 | 利润弹性强，但现金流约束大 | Capex、营运资本、终值、回款是否吞噬利润 | Capex、D&A、存货、应收、OCF/净利 |
| PEG 低、DCF 高 | 成长斜率一般，但现金流质量好 | 是否是稳态现金牛、或市场低估现金转化 | 收入增速、订单连续性、分红/回购、FCF 收益率 |
| PEG 低、DCF 低 | 利润弹性和现金流支撑都弱 | 主线是否降级，估值是否只能靠情绪 | 利润下修、毛利率、现金流、订单证伪 |

## Difference Attribution

分歧必须拆到具体来源：

| 分歧来源 | 解释方式 |
| --- | --- |
| 收入增速 | PEG 对增速敏感，DCF 对增速持续性和现金转化敏感 |
| 毛利率/费用率 | PEG 可能奖励利润率修复，DCF 要看是否可持续 |
| Capex | PEG 不直接惩罚扩张投入，DCF 会扣减 UFCF |
| 营运资本 | PEG 容易忽略应收/存货，DCF 会体现现金占用 |
| 终值 | DCF 若过度依赖终值，说明显性期现金流支撑不足 |
| WACC | 高风险公司 DCF 折现率上升，PEG 可能仍受市场风险偏好支撑 |
| 年份切换 | PEG 可提前交易远期利润，DCF 需要现金流路径闭合 |
| 证据等级 | 弱证据可影响 PEG 情绪上沿，但不能支撑 DCF 基准 |

## Aggregation Rule

不要机械平均 PEG 与 DCF。聚合报告应写成：

```text
PEG 给出成长定价区间；
DCF 给出现金流内在价值区间；
二者差异说明市场对未来验证的依赖程度；
跟踪指标决定后续应重点观察哪组模型假设。
```

当二者严重分歧：

- 不得用一句“取中值”解决。
- 不得创造一个新的综合目标价覆盖上游模型。
- 只列出 3-5 个能让分歧收敛的财报或经营指标。
