# Wayne's Skill

Personal Codex skills maintained as versioned assets.

## Skills

- `local-bank-research`: 地方商业银行股投研框架，覆盖利润桥、净利息收入、资产质量、债券投资、区域 beta、分红与预期差。

## Local install

Install or update a skill from this repo into Codex:

```bash
mkdir -p ~/.codex/skills
cp -R skills/local-bank-research ~/.codex/skills/local-bank-research
```

Restart Codex after installing or updating skills.
