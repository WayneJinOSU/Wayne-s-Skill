# Wayne's Skill

Personal Codex skills maintained as versioned assets.

## Skills

- `doc`: Office documents, PDFs, presentations, and spreadsheets utilities.
- `find-skills`: Discover and install useful Codex skills.
- `frontend-design`: Distinctive frontend UI design guidance.
- `local-bank-research`: 地方商业银行股投研框架，覆盖利润桥、净利息收入、资产质量、债券投资、区域 beta、分红与预期差。
- `pipecat-deploy`: Deploy Pipecat agents.
- `pipecat-init`: Scaffold Pipecat projects.
- `pipecat-talk`: Start Pipecat voice conversations.
- `skill-github-sync`: Sync local Codex skills into this GitHub repository.
- `wencai-query`: 同花顺问财查询与 A 股筛选分析。

## Local install

Install or update a skill from this repo into Codex:

```bash
mkdir -p ~/.codex/skills
cp -R skills/* ~/.codex/skills/
```

Restart Codex after installing or updating skills.
