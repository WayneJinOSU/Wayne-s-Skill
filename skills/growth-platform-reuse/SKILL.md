# 平台复用能力研究

## Purpose

用于回答：新业务是不是靠原有能力自然切入，而不是硬蹭概念？

## Inputs

优先读取公司原有产品技术、客户结构、工艺路线、产线/产能、供应链、认证体系、研发团队和新产品说明。

## Workflow

1. 列出旧业务的核心能力。
2. 列出新业务需要的关键能力。
3. 逐项匹配技术、客户、工艺、产能、供应链、认证体系。
4. 判断是否存在完全跨界、能力断层或概念包装。
5. 说明复用链条如何降低新业务兑现难度，并列出替代解释。
6. 给出可信度等级：高复用、中复用、弱复用、硬跨界。
