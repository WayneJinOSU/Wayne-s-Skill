# 利润桥研究

## Purpose

用于回答：新故事能不能转化为未来利润中枢？未来利润能否消化当前估值？

## Inputs

优先读取历史利润、业务分部毛利率、费用率、产能、订单、价格、出货量、资本开支、折旧、研发费用、财务费用和管理层指引。

## Formula

```text
未来利润 =
存量业务利润
+ 新业务新增利润
- 扩张期费用/折旧/财务成本/爬坡损耗
```

## Workflow

1. 建立存量业务利润底盘。
2. 估算新业务收入：TAM、份额、价值量、产能、订单或出货。
3. 估算新业务利润率：毛利率、净利率、规模效应、议价能力。
4. 估算扩张成本：研发、折旧、人员、销售、财务、汇兑和爬坡损耗。
5. 做保守、中性、乐观三种情景。
