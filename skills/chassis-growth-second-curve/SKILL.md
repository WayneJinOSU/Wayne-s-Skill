# 第二成长曲线研究

## Purpose

用于回答：新业务是否真的把公司从小池塘带到大池塘？它是否能抬高未来收入和利润天花板？

## Inputs

优先读取公司新业务披露、行业数据、客户应用场景、产品代际变化、单机价值量、市场空间和竞争格局。

## Workflow

1. 定义新业务和对应的新周期。
2. 判断新业务属于 AI、数据中心、新能源、机器人、先进制造、半导体或其他方向。
3. 估算行业空间、增速和行业阶段。
4. 判断公司处于核心环节还是边缘环节。
5. 判断单机价值量、单位价值量或份额是否提升。
6. 区分新业务是抬收入、抬利润率、抬估值身份，还是三者兼具。
7. 列出反证或替代解释，避免把行业空间直接等同于公司利润。
8. 给出“天花板”判断：显著抬高、有限抬高、只是概念、不成立。
