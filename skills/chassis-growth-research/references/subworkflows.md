# 主工作流与子工作流

## 执行原则

每次分析必须先跑主工作流，再逐个完成子工作流 A-H。每个子工作流都要输出：

- 结论：成立、部分成立、不成立、证据不足
- 核心证据：尽量引用财报、公告、交易所互动、公司官网、投资者关系纪要、可靠行情数据
- 不确定性：哪些信息缺失或可能改变判断
- 对最终报告的影响：提高确信度、降低确信度、提高估值弹性、提高证伪风险

不要在完成子工作流前给最终判断。最后的结论必须能回溯到每个子工作流的小结。

## 主工作流：适用性判断

目标：判断公司是否适合“底盘型成长股”框架。

必须回答：

- 是否有旧业务底盘
- 是否有 AI、数据中心、新能源、机器人、先进制造等第二成长曲线
- 新业务是否能复用公司原能力
- 是否已有承接动作
- 是否属于不适用行业或纯题材公司

输出格式：

| 项目 | 判断 | 证据 | 影响 |
| --- | --- | --- | --- |
| 旧业务底盘 | 成立/部分/不成立 |  |  |
| 第二成长曲线 | 成立/部分/不成立 |  |  |
| 平台复用 | 成立/部分/不成立 |  |  |
| 承接动作 | 成立/部分/不成立 |  |  |
| 框架适用性 | 适用/部分适用/不适用 |  |  |

## 子工作流 A：底盘是否够硬

目标：判断公司下限。

检查项：

- 原有业务收入规模和增速
- 原有业务利润和毛利率
- 经营现金流和应收/存货压力
- 客户基础和行业地位
- 业务是否足够分散
- 没有新业务时公司是否仍能维持合理盈利

输出小结：

```text
底盘结论：
核心证据：
主要风险：
对估值的影响：
```

## 子工作流 B：新赛道是否抬高天花板

目标：判断公司上限。

检查项：

- 新业务是什么
- 对应市场空间和未来 3-5 年增速
- 行业处于早期、中期还是成熟期
- 技术路线是否清晰
- 公司处于核心环节还是边缘环节
- 单机价值量或单位价值量是否提升

输出小结：

```text
天花板结论：
新业务：
行业空间：
公司位置：
关键不确定性：
```

## 子工作流 C：平台复用能力

目标：判断可信度。

检查项：

- 技术是否复用
- 工艺是否复用
- 客户是否复用
- 供应链是否复用
- 产能是否复用
- 认证体系是否复用
- 是否存在完全跨界或概念化包装

输出小结：

```text
复用结论：
复用链条：
跨界风险：
可信度：
```

## 子工作流 D：承接动作

目标：判断预期能否落地。

检查项：

- 扩产公告
- 资本开支
- 定增、H 股、可转债、项目贷款
- 海外基地
- 设备采购
- 客户验证或定点
- 订单释放
- 量产进度
- 研发投入和人员扩张

输出小结：

```text
承接结论：
已发生动作：
尚未验证事项：
落地风险：
```

## 子工作流 E：成长阶段

目标：判断市场交易的是预期差、兑现，还是透支。

分类：

- 预期型：新业务刚被发现，利润尚未明显体现
- 半兑现型：新业务已放量，市场验证持续性
- 兑现型：业绩大幅兑现，市场充分定价
- 不符合：没有底盘或没有真实新曲线

检查项：

- 新业务是否已有收入或订单
- 利润是否已经体现
- 股价是否已提前反映
- 市场仍在担心什么
- 财报是否证伪或验证市场担忧

输出小结：

```text
阶段结论：
为什么不是上一阶段：
为什么还没到下一阶段：
股价弹性来源：
```

## 子工作流 F：利润桥

目标：把故事转成利润中枢。

固定公式：

```text
未来利润 =
存量业务利润
+ 新业务新增利润
- 扩张期费用/折旧/财务成本/爬坡损耗
```

拆解要求：

- 存量业务利润：用历史利润、毛利率、费用率建立底盘
- 新业务收入：用 TAM、市占率、价值量、产能或订单估算
- 新业务利润率：考虑毛利率、净利率、规模效应、议价能力
- 扩张期成本：研发、折旧、人员、销售、财务、汇兑、产能爬坡

## 子工作流 G：远期估值

目标：判断当前市值是否透支。

要求：

- 使用最新股价和总市值，并注明日期
- 用 2025、2026E、2027E、2028E 利润假设计算 PE
- 明确利润假设是保守、中性还是乐观
- 说明估值能否被未来利润消化

## 子工作流 H：证伪点

目标：找出最可能推翻逻辑的变量。

至少覆盖：

- 新业务订单或客户验证
- 毛利率和净利率
- 产能爬坡和折旧压力
- 行业技术路线变化
- 竞争格局恶化
- 当前估值是否已经透支
- 现金流、应收账款、存货等财务质量

输出要求：

按重要性列 3-5 个证伪点，并说明应该跟踪什么数据。

## 最终报告合成

最终报告不要简单拼接子工作流。需要给出一张“子工作流结论表”，再合成：

- 一句话结论
- 是否适用框架
- 当前成长阶段
- 最大预期差
- 利润桥和估值表
- 最关键证伪点
- 对不同类型资金的适配度

建议结论语言：

```text
这家公司不是因为当期便宜而有吸引力，而是因为 X 子工作流和 Y 子工作流显示利润中枢可能上移；但 Z 子工作流显示当前估值已经计入部分预期，因此后续股价弹性取决于 A/B/C 是否继续兑现。
```
