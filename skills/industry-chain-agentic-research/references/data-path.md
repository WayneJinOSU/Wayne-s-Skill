# 数据路径与证据优先级

## 证据优先级

1. 官方与半官方数据：统计局、发改委、工信部、能源局、海关、交易所、行业协会、国际组织。
2. 公司公告原文：年报、季报、招股书、募集说明书、重大合同、订单、调研纪要、投资者关系。
3. 结构化数据：`$tushare`、`$wencai-query`、行情估值、财务三表、同业筛选、概念和市场排名。
4. 产业数据：价格、库存、开工率、排产、招标、订单、运价、装机、销量、渗透率。
5. 券商、媒体和市场口径：只能作为观点、预期或交叉验证，不能替代事实。

当公告与市场口径冲突时，以公告或官方数据为准，并说明口径差异。

## 按模块取数

| 模块 | 优先数据 | 辅助数据 |
| --- | --- | --- |
| 行业定义 | 行业标准、协会分类、公司业务说明 | 新闻、券商框架 |
| 产业链 | 年报业务链条、招股书、协会资料 | 公司官网、调研纪要 |
| 需求 | 终端销量、渗透率、capex、出口、替换周期 | 订单、招标、渠道反馈 |
| 供给 | 产能、开工率、库存、建设周期、退出政策 | 排产、价格、龙头扩产公告 |
| 价格 | 现货/期货价格、招投标价、批价、ASP | 渠道报价、第三方数据库 |
| 利润池 | 分环节毛利率、净利率、ROIC、现金流 | 同业财务、成本曲线 |
| 竞争格局 | CR3/CR5、份额、客户认证、专利、牌照 | 供应链名单、客户切换案例 |
| 财务映射 | 收入、毛利率、费用率、现金流、应收、存货、资本开支 | 价格和销量敏感性 |
| 估值 | PE/PB/PS/EV/EBITDA/PFCF、历史分位、同行估值 | 市场一致预期、盈利预测 |
| 风险证伪 | 高频指标和财报指标 | 反向案例、政策技术变化 |

## 输出口径

每份报告至少说明：

- 数据查询日期。
- 哪些结论来自事实，哪些是推断，哪些是情景假设。
- 关键数据的来源和口径。
- 证据缺口如何影响结论强度。

## 数据不足时

不要编造数据。必须写：

```text
证据不足：
缺少的数据：
当前只能做出的降级判断：
后续应跟踪的数据：
```
